Schedulist Style Guide

Fonts

We'll be utilizing Google Web Fonts' Open Sans at weights 300, 400, 600, and 700. While the mockups use Museo Sans, which is slightly more refined and stylized, Open Sans is a worthy comparison. 

<link href='http://fonts.googleapis.com/css?family=Open+Sans:300,400,600,700' rel='stylesheet' type='text/css'/>

Buttons

To apply a button style to any element (namely a and button tags) simply add a class="btn". Modifiers can be used to add color, or state, such as .disabled.

Example

Button (a.btn)

Button (a.btn.blue)

Button (a.btn.red)

Button (a.btn.disabled)

Usage

<button class='btn'>Button (button.btn)</button>
<a href="" class='btn blue'>Button (a.btn.blue)</a>
